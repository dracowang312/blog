关于.net全栈工程师

2024年需要全栈工程师，但是学习.net出来不一定要从事互联网行业

全栈开发工程师需要会的技能

  • 前端：用户能看到的界面;交互（用户与服务器）
  • 服务器：数据库操作；给前端提供数据服务
  • 数据库：功能是存储数据

Java/python/Node.js比较

Java

应用：

  • 企业级开发/银行/铁路/大型应用
  • 移动端： android
  • 大数据开发

特点：

  • 优点；应用广/开源/现有框架多
  • 不足：难度/技术更新快/不适合做桌面应用/就业要求（学历要求）

PHP

应用：网站
特点：

  • 优点：简单
  • 不足：安全，性能

python

应用：Al和大数据开发

.net（c#）

应用：

  • 桌面应用（强项：上位机（c/c++/c#/QT））
    • 制造业
    • 传统工业

特点：

  • 优点： 对学历要求不高
  • web应用
    • 政府的一些项目（钱好拿，性能要求不高）
    • 互联网（基本被JAVA吃光了，但是可以做）
    • MES系统（制造执行系统）
  • 大数据

web应用(b/s)与桌面应用(c/s)

web应用(b/s)：browser/server

  • 特点：同桌面相反

c/s: client/server

  • 优点：安全/性能
  • 不足：下载安装/更新

就业条件

需要具备的技术

前端技术

  • HTML+CSS:静态界面
  • JavaScript语言：交互界面（动）
  • 前端框架（vue/react/Angular）
  • 小程序（微信小程序）

服务器开发技术

  • c#语言
  • ADO.net与EF：操作数据库
  • Linq：对象操作技术
  • 桌面应用开发（winform + WPF）
  • web应用开发(Asp.net MVC)

数据库技术

  • MS SQLServer
  • MySQL｜PostgreSQL(开源)

辅助技术

  • Git
  • 原型｜PS

上位机

  • C#/ADD.NET/EF
  • 多线程开发，通讯协议
  • winform ->WPF(界面更漂亮)

MES

  • ASP.NET
  • 前端技术

软技能

  • 学习能力
  • 抗压能力
  • 沟通能力
  • 学习能力
